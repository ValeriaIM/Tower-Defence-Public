# Tower defence

ЯТП игра мы сделали:

* основные классы(Башня, Враг, Волна, Игрок, Карта)
* дорогa
* взаимодействие врагов и башен
* удаление врага из процесса игры
  
что нужно:

* магазин башен

## Дополнительно
Карта обозначения: (way1)

* B - точка начала пути монстров
* E - точка конца пути монстров
* '-' движ-е вправо
* '|' движ-е вниз
* '.' место для башен
